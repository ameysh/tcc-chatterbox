# TCC Chatterbox Discord Bot

A custom Discord bot powered by Ollama Generative AI and built for the TCC Gamerforge private Discord server. This bot is designed specifically for our community and is not intended for public use or distribution.

## About This Project

This is a **private Discord bot** created for personal use within our closed Discord community. The code is made public, but the bot itself is not available for public installation or use.

## Features

- **AI-Powered Responses**: Integrated with Ollama for AI-powered chat responses
- **Manual Message Broadcasting**: Send messages as the bot through terminal commands
- **Slash Commands**: Modern Discord slash command support
